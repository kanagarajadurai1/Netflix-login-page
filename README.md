# 🎬 Netflix Login Page Clone

> A **Netflix-inspired full-stack web app** built with React + Vite and deployed on Vercel.

---

## 🚀 Live Demo

🌐 **Frontend:** [netflix-login-page.vercel.app]( https://netflix-login-page-sigma.vercel.app/)

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 🔐 Login Auth | Email & password validation |
| 🎥 Movie Dashboard | Trending, Action, Drama & Comedy rows |
| 🔍 Live Search | Filter movies instantly by title or genre |
| 🎞️ Movie Modal | Backdrop, rating, genre & description popup |
| 🎨 Netflix UI | Animated poster grid, floating labels, red glow |
| 📱 Responsive | Works perfectly on mobile & desktop |

---

## 🛠️ Tech Stack

```
Frontend  →  React + Vite + CSS
Backend   →  Node.js + Express
Deploy    →  Vercel
Data      →  TMDB Poster Images
```

## ⚙️ Run Locally

```bash
# 1️⃣ Clone the repo
git clone https://github.com/kanagarajadurai1/Netflix-login-page.git
cd Netflix-login-page

# 2️⃣ Start Backend
cd backend
npm install
node server.js

# 3️⃣ Start Frontend
cd ../frontend
npm install
npm run dev

# 4️⃣ Open in browser
http://localhost:5173
```
